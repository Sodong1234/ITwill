# [오전수업]

## table을 만드는 방법

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>html1/test4.html</title>
</head>
<body>
<h2>표만들기</h2>
<table border="1">
<tr><td>1행1열</td><td>1행2열</td></tr>
<tr><td>2행1열</td><td>2행2열</td></tr>
</table>
</body>
</html>
```

table를 만들때는 table, tr, td 코드를 사용함
- tr은 행(가로), td는 열(세로)
- table 옆에 붙이는 border는 선의 크기(개수)를 나타냄.



### 5행 4열 표 만들기
```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>html1/test4.html</title>
</head>
<body>
</table>
<h2>5행 4열 표 만들기</h2>

<tr><td>가정용</td><td>5kg</td><td>18-26과</td><td>47000원</td></tr>

```
